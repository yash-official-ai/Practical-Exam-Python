# 📚 E-Library Data Insights Dashboard

## 📖 Project Overview
The **E-Library Data Insights Dashboard** is a Python-based data analysis project developed using **OOP, Pandas, NumPy, Matplotlib, and Seaborn**. It analyzes library transaction records and provides useful insights through statistics, filtering, reports, and visualizations.

---

## ✨ Features
- 📂 Load CSV Dataset
- 🔍 Explore Library Data
- 📊 Calculate Statistics
- 🎯 Filter Transactions
- 📑 Generate Summary Report
- 📈 Data Visualization
  - Bar Chart
  - Line Graph
  - Pie Chart
  - Heatmap

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📁 Dataset
**File:** `library_transactions.csv`

### Dataset Columns
- Transaction ID
- Date
- User ID
- Book Title
- Genre
- Borrowing Duration (Days)
